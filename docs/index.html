Features

Everything a bug bounty hunter needs

Built specifically for the bug bounty workflow. Not a port scanner. Not a DAST tool. A complete semi-automated hunting platform.

🔍
Full Recon Pipeline
One-click full recon chains subfinder, amass, httpx, nmap, katana, gau, waybackurls, and gowitness into a single automated workflow.
🎯
CVE Correlation
Detected technologies (nginx, WordPress, Spring...) are automatically matched to known CVEs. Run targeted nuclei scans per tech stack with one click.
📸
Visual Host Triage
gowitness automatically screenshots every live host. Browse thumbnails to quickly spot admin panels, login pages, and interesting targets.
🛡️
Security Checks
CORS misconfigs, 403 bypass (20+ techniques), cloud bucket exposure, GitHub secret scanning, and OOB callbacks via interactsh.
🔀
Proxy + Repeater
Intercept live traffic, edit and replay requests, and manage session cookies and headers globally across all tools.
📁
Project Management
Full project isolation. Findings, recon data, notes, and session data are scoped per target. Switch between programs instantly.
💾
Findings Database
Structured storage for every vulnerability: severity, evidence, status, and notes. Everything you need to write the report.
🔄
Auto-update
NexHunt checks for new releases on startup. Update with one click — no manual downloads, no broken installs.
