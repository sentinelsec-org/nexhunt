Installation

One command to get started

Installs all required tools, builds the app, and sets up your environment. Kali Linux and Debian-based distros supported.

bash
curl -fsSL https://github.com/sentinelsec-org/nexhunt/releases/download/v1.1.0/nexhunt-1.1.0.tar.gz | tar xz && sudo bash install.sh
