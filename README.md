## What is NexHunt?

NexHunt is a **desktop app for bug bounty hunters and penetration testers** that replaces your messy collection of terminal tabs, scripts, and notes with a single, integrated workflow.

From the moment you create a project, NexHunt guides you through the full attack surface вҖ” from subdomain discovery all the way to exploiting findings and generating reports. Every tool's output feeds the next phase automatically. You find more, faster.

It runs **locally on your Linux machine**. No cloud, no data sent anywhere (except the PRO AI Copilot, which routes through Sentinel's privacy-respecting proxy). Your findings stay yours.

## рҹ—әпёҸ The full workflow, in one app

### рҹ”Қ Reconnaissance

The most complete recon pipeline you can run from a single button.

- **Subdomain enumeration** вҖ” subfinder + amass running in parallel, results merged and deduplicated automatically
- **Live host probing** вҖ” httpx identifies status codes, technologies, titles, and response fingerprints across all discovered hosts
- **Port scanning** вҖ” nmap on all live hosts, with service detection
- **Web crawling** вҖ” katana and linkfinder extract every endpoint, JS file, and form from live targets
- **URL history** вҖ” gau + waybackurls pull years of archived URLs from Wayback Machine and Common Crawl
- **Parameter discovery** вҖ” paramspider and arjun find hidden HTTP parameters across all collected URLs
- **Endpoint analysis** вҖ” analyzes discovered endpoints for interesting patterns, auth requirements, and potential attack surface
- **Screenshots** вҖ” gowitness automatically screenshots every live host so you can triage visually

> рҹ’Ў The **Full Recon** mode chains all of the above into one pipeline that runs while you grab a coffee.

---

### рҹҺҜ Vulnerability Scanning

Find what's actually exploitable, not just what's detectable.

- **Nuclei** вҖ” 8,000+ community templates covering CVEs, misconfigurations, exposures, default credentials, and more. NexHunt pre-loads **OWASP Top 10 presets** and lets you run targeted scans per technology
- **CVE Correlation** вҖ” automatically matches discovered technologies (nginx, Apache, WordPress, Spring...) to known CVEs and runs targeted nuclei templates. One click per tech stack
- **Directory brute-force** вҖ” ffuf, gobuster, and dirsearch with smart wordlist selection based on detected server type
- **Web server audit** вҖ” nikto catches misconfigurations, outdated headers, and known server vulnerabilities

---

### рҹ’Ҙ Exploitation

Validate findings. Prove impact. Write better reports.

- **SQLi** вҖ” sqlmap on discovered parameters, automatically handles WAF evasion, tamper scripts, and DB enumeration
- **XSS** вҖ” dalfox (fast, accurate) and xsstrike (deep DOM analysis) on all collected endpoints
- **Command injection** вҖ” commix on forms and parameters with command injection indicators
- **SSRF / Open redirect** вҖ” interactsh-backed testing for out-of-band vulnerabilities

---

### рҹӣЎпёҸ Security Tools

Specialized checks that most scanners miss.

- **CORS misconfiguration** вҖ” tests origin reflection, credential exposure, and wildcard policies across all live hosts
- **403 Bypass** вҖ” tries 20+ bypass techniques (header injection, path normalization, verb tampering) on forbidden endpoints
- **Cloud bucket exposure** вҖ” discovers misconfigured S3, GCS, and Azure Blob buckets associated with the target
- **GitHub secret scanning** вҖ” TruffleHog scans the target org's public repositories for leaked credentials, API keys, and tokens
- **OOB interaction** вҖ” interactsh listener for DNS/HTTP callbacks from blind injection points

---

### рҹ”Җ Proxy & Traffic Analysis

Your HTTP Swiss knife, built into the workflow.

- **Live traffic capture** вҖ” intercept and inspect every request from your browser through NexHunt's proxy
- **Request editor** вҖ” modify and replay any captured request with full header and body control
- **HTTP Repeater** вҖ” save interesting requests and replay with modifications
- **Proxy Intruder** *(PRO)* вҖ” automated fuzzing with cluster bomb and pitchfork attack modes, payload wordlists, and response filtering

## вҡЎ Installation

### One-liner

```bash
curl -fsSL https://github.com/sentinelsec-org/nexhunt/releases/download/v1.1.0/nexhunt-1.1.0.tar.gz | tar xz && sudo bash install.sh
```

The installer:
- Installs all 20+ security tools (nmap, nuclei, subfinder, ffuf, sqlmap, dalfox, gowitness, katana, gau, waybackurls, gobuster, nikto, dirsearch, commix, arjun, paramspider, xsstrike, amass, httpx, interactsh...)
- Sets up the Python backend in an isolated venv
- Builds the Electron frontend
- Adds `nexhunt` to your PATH
- Creates a `.desktop` entry for your app launcher

### Requirements

| Requirement | Notes |
|---|---|
| Linux | Kali, Debian, Ubuntu вҖ” tested on Kali 2024+ |
| Python 3.10+ | Available on all supported distros |
| Node.js 18+ | Installed automatically if missing |
| Go 1.21+ | Installed automatically if missing |
| ~2 GB disk | For all tools + venv + build |
| Internet | For initial install only |

### Update

```bash
sudo bash install.sh --update
```

Or from inside the app: **Settings вҶ’ Updates вҶ’ Check for updates**.
